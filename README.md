# Placement-Eligibility-Checker
A Python application to check placement eligibility based on academic criteria.
## About
This project is a simple Python application that checks whether a student is eligible for placement based on academic details such as CGPA, SSC percentage, Intermediate percentage, and active backlogs.

The project was created to practice Python programming by solving a real-world student problem.

## Features

- Enter student details
- Check placement eligibility
- Display reasons if not eligible
- Simple console-based interface

## Technologies Used

- Python 3
